😠 ANIMATION DE COLÈRE (Touche C)
═══════════════════════════════════════════════════════════════

Quand vous appuyez sur C :

   1. Les yeux deviennent ROUGES 🔴
   2. Ils pulsent en croix :
      • S'élargissent horizontalement
      • Se rétrécissent verticalement
      • Puis inversement
   3. Animation continue et fluide
   4. Très visible et impressionnant !

Paramètres de l'animation :
   • Fréquence : 15 (vitesse du pulse)
   • Amplitude : 8 pixels
   • Couleur : RGB(255, 50, 50) - Rouge vif

💡 DÉTAILS TECHNIQUES
═══════════════════════════════════════════════════════════════

Animation de Colère :
   Fichier : roboeyes_pygame.py
   Méthode : _handle_angry()
   Couleur : self.angry_color = (255, 50, 50)
   Pulse   : width += pulse, height -= pulse/2

Menu Amélioré :
   Fichier : robot_eve_pro.py
   Fonction : draw_icon_button()
   Taille emoji : 42px (au lieu de 36px)
   Taille bouton : 70x60 (au lieu de 60x50)
   Labels : Ajoutés pour clarté
