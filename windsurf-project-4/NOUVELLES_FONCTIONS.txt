───────────────────────────────────────────────────────────────

⏱️  RESET AUTOMATIQUE
═══════════════════════════════════════════════════════════════

Après chaque émotion, le robot revient automatiquement
à l'état NORMAL après 3 secondes !

Comment ça marche :
   1. Appuyez sur une touche (L, C, D, T, P, R)
   2. L'émotion s'affiche
   3. Attendez 3 secondes
   4. ➡️ Reset automatique vers Normal
   5. Message dans la console

Pour annuler le reset :
   • Appuyez sur 0 (zéro) pour rester en Normal
   • Ou appuyez sur une autre émotion

🧪 MODE TEST AUTOMATIQUE
═══════════════════════════════════════════════════════════════

Touche : M

Active un mode qui teste TOUTES les émotions automatiquement !

Séquence de test (2 secondes chacune) :
   1. 💛 Sourire (jaune avec rayons)
   2. 🔴 Colère (rouge qui pulse)
   3. 💚 Dégoût (vert avec arcs)
   4. 💙 Tristesse (bleu)
   5. 💧 Pleurs (bleu avec larmes)
   6. 😂 Rigolade (bleu qui tremble)
   7. 😐 Normal (bleu)
   → Recommence en boucle

Utilisation :
   • Appuyez sur M pour activer
   • Le texte devient VERT en bas
   • Chaque émotion s'affiche 2 secondes
   • Appuyez à nouveau sur M pour désactiver

Parfait pour :
   ✅ Tester toutes les émotions rapidement
   ✅ Vérifier que tout fonctionne
   ✅ Démonstration automatique
   ✅ Validation du robot

🎮 NOUVELLES TOUCHES
═══════════════════════════════════════════════════════════════

ÉMOTIONS :
   L  → 💛 Sourire (JAUNE avec rayons) ⭐ NOUVEAU !

   R  → 😂 Rigolade (bleu qui tremble)

CONTRÔLES :
   O  → Ouvrir les yeux
   F  → Fermer les yeux
   B  → Cligner

   M  → Mode Test (ON/OFF) ⭐ NOUVEAU !
   ESC → Quitter
