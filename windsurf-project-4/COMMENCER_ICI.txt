📋 ÉTAPES RAPIDES (10 minutes)
═══════════════════════════════════════════════════════════════

1️⃣  INSTALLER LES DÉPENDANCES
   
   Sur Linux/Mac :
   ./install.sh
   
   Sur Windows :
   install.bat
   
   Ou manuellement :
   pip install pygame numpy

───────────────────────────────────────────────────────────────

2️⃣  LANCER LE PROGRAMME PRINCIPAL
   
   Méthode 1 (Recommandée) :
   ./run_robot.sh              # Linux/Mac
   python3 run_robot.py        # Tous systèmes
   
   Méthode 2 (Directe) :
   .venv/bin/python robot_smile.py    # Si vous avez un venv
   python3 robot_smile.py             # Linux/Mac
   python robot_smile.py              # Windows

───────────────────────────────────────────────────────────────

3️⃣  INTERAGIR AVEC LE ROBOT
   
   ESPACE  →  Faire sourire le robot 😊
   O       →  Ouvrir les yeux 👀
   C       →  Fermer les yeux 😴
   B       →  Cligner 😉
   ESC     →  Quitter

🛠️  MATÉRIEL POUR ROBOT RÉEL (optionnel)
═══════════════════════════════════════════════════════════════

Pour créer un vrai robot physique :

  • Raspberry Pi Pico ou ESP32        (~5-8€)
  • Écran OLED I2C 128x64 (SSD1306)   (~10€)
  • Câbles de connexion                (~3€)
  • Batterie USB (optionnel)           (~10€)

  📖 Voir micropython/README.md pour les détails
