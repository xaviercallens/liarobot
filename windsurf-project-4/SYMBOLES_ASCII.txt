📖 GUIDE D'UTILISATION
═══════════════════════════════════════════════════════════════

1. Regardez le menu en bas
2. Trouvez le symbole ASCII (ex: :-) pour sourire)
3. Lisez le label sous le symbole
4. Appuyez sur la touche indiquée (en jaune)
5. L'émotion s'affiche !

Exemple :
   Je veux que le robot soit en colère
   → Je cherche >:( dans le menu
   → Je vois "Colere" en dessous
   → J'appuie sur C (en jaune)
   → Les yeux deviennent ROUGES et pulsent !

🎯 CORRESPONDANCE SYMBOLES / ÉMOTIONS
═══════════════════════════════════════════════════════════════

:-)  = Sourire classique (happy)
>:(  = En colère (angry) → YEUX ROUGES !
:-P  = Tire la langue (disgust)
:-(  = Triste (sad)
:'(  = Pleure (crying) → avec larmes
:-D  = Rire (laughing) → yeux tremblent
:-|  = Neutre (normal)

O_O  = Yeux ouverts
-_-  = Yeux fermés
;-)  = Clin d'œil
